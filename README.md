### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/No-Country/C6-47-t-ReactNative.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Fill your `.env`
   ```js
    HTTPPORT=
    HTTPSPORT=

    DB_NAME=
    DB_USER=
    DB_PASSWORD=
    DB_HOST=
    DB_PORT=

    JWT_SECRET=
   ```
